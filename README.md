# Python Selenium Study Repository

This repository is a personal study project focused on learning Python automation and Selenium WebDriver.

It is organized as a course-based practice archive, where each folder contains small exercises, examples, and notes that help reinforce the concepts being studied.

## Repository structure

- Python Basics: foundational Python concepts such as variables, lists, dictionaries, loops, functions, exceptions, and object-oriented programming.
- SeleniumWDTutorial: Selenium automation examples covering browser interactions, locators, waits, frames, file uploads, and other practical WebDriver topics.
- Notes and Resources: additional learning notes and supporting material.

## Purpose

The goal of this repo is simple: keep a clean, searchable collection of lessons and hands-on practice while progressing through the course.

## Notes

This is a study repository, so the content is intentionally practical and educational rather than production-oriented.
